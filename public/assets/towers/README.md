# Tower Assets

Place tower PNG images here. The game will automatically load them.
If an image is missing, the procedural fallback will be used.

## Image Size
Recommended: **200x200 pixels** (will be scaled to 70px in-game)

## File Names

### Stone Age (Era 0)
- `stone_basic.png` - Slinger
- `stone_rapid.png` - Hunter
- `stone_sniper.png` - Spearman
- `stone_aoe.png` - Rock Thrower

### Castle Age (Era 1)
- `castle_basic.png` - Longbow
- `castle_rapid.png` - Crossbow
- `castle_sniper.png` - Ballista
- `castle_aoe.png` - Mangonel
- `castle_laser.png` - Mage Tower
- `castle_frost.png` - Frost Mage

### Imperial Age (Era 2)
- `imperial_basic.png` - Sentry Gun
- `imperial_rapid.png` - Gatling
- `imperial_sniper.png` - Sniper
- `imperial_aoe.png` - Mortar
- `imperial_laser.png` - Laser Cannon
- `imperial_frost.png` - Cryo Ray
- `imperial_shock.png` - Tesla Tower
- `imperial_missile.png` - Rocket Silo
